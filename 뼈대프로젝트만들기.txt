<뼈대 프로젝트 만들기 과정>
1.django-admin startproject [프로젝트명]
2.python manage.py startapp [앱(서브프로그램)명]
3.setting.py 설정.(host,apps,db,시간 등)
4.python manage.py migrate	//db 기본 테이블 생성
5.python mange.py runserver 	//서버 실행.
  ex) python mange.py runserver 0:8000

*관리자(슈퍼유저) 계정 생성
 : python manage.py createsuperuser 
  -> 아이디,이메일,비번 설정
